# Use Cases проекта «КиноBooru»

## UC-01 Просмотр списка фильмов

**Актор:** Гость  

**Предусловия:**  
- Пользователь открыл сайт

**Основной сценарий:**
1. Пользователь переходит на главную страницу
2. Система отображает список доступных фильмов
3. Пользователь может пролистывать страницы списка

**Альтернативные сценарии:**
- Фильмов нет — отображается пустой список

**Результат:**  
Пользователь видит список фильмов

---

## UC-02 Просмотр подробной информации о фильме

**Актор:** Гость  

**Предусловия:**  
- Фильм существует в системе

**Основной сценарий:**
1. Пользователь выбирает фильм из списка
2. Система отображает страницу фильма
3. Отображаются описание, постер, видео, теги и отзывы

**Альтернативные сценарии:**
- Фильм не найден — возвращается ошибка 404

**Результат:**  
Пользователь получает полную информацию о фильме

---

## UC-03 Регистрация пользователя

**Актор:** Гость  

**Предусловия:**  
- Пользователь не авторизован

**Основной сценарий:**
1. Пользователь открывает форму регистрации
2. Вводит имя пользователя, email и пароль
3. Отправляет форму
4. Система создаёт нового пользователя

**Альтернативные сценарии:**
- Некорректные данные — отображается сообщение об ошибке
- Пользователь с таким именем уже существует

**Результат:**  
Пользователь зарегистрирован в системе

---

## UC-04 Авторизация пользователя

**Актор:** Пользователь  

**Предусловия:**  
- Пользователь зарегистрирован

**Основной сценарий:**
1. Пользователь вводит логин и пароль
2. Система проверяет данные
3. Система выдаёт JWT токен

**Альтернативные сценарии:**
- Неверный логин или пароль

**Результат:**  
Пользователь авторизован

## UC-08 Создание фильма (модератор)

**Актор:** Модератор  

**Предусловия:**  
- Пользователь имеет роль модератора

**Основной сценарий:**
1. Модератор вводит данные фильма
